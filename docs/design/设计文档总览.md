## 3. 当前产品目标摘要

- 减少手机使用时间
- 建立统一、优雅、精致的像素体验
- 用像素风格去除现代 UI 的繁杂
- 通过 Home 和 Idle 减少为了确认状态而打开 App 的次数

## 4. 当前技术基线摘要

- 单 `Activity`
- 单一 `LauncherState`
- 自定义像素渲染链路
- 主要代码目录为 `app/src/main/kotlin`
- 抽屉和设置页已经共享文本列表基座

## 5. 文档维护约定

- 只要代码目录、状态模式、核心交互或数据源发生变化，必须同步更新第二层技术文档
- 只要产品目标、模块职责或验收标准发生变化，必须同步更新第三层产品文档
- 不允许让设计文档长期假装“当前实现已经等于目标”
- 文档中引用代码路径时，统一使用 `app/src/main/kotlin/...`
